BLOGGER PROFILE by Ceardach

A feature module based on the Kit specifications (http://drupal.org/project/kit)

Provides the following functionality:

  * Profile fields for those with the 'blogger' role
    - Real name
    - Information with optional summary

  * Displays blogger information on the user's profile page

  * Displays an 'about the author' block on the first sidebar


Required contributed modules:

  * features
  * views
  * context
  * strongarm
  * ctools
  * token
  * profile2
  * realname


Required feature module:

  * ceardach_blog (https://github.com/ceardach/ceardach_blog)
    Provides the base functionality for creating a blog
